\# Projeto SQL - Modelo iFood



Este banco de dados foi desenvolvido como uma atividade prática na aula de Banco de Dados 1 no CEUB.



O objetivo foi modelar um sistema de entregas, criando tabelas para clientes, entregadores e pedidos. No script, usei comandos para criar a estrutura (DDL) e também para manipular os dados (DML), como os exercícios de UPDATE e DELETE que fiz para testar a gestão das tabelas.



Ferramenta: MySQL Workbench.
