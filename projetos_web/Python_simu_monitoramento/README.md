\# Monitoramento de Satélite



Este código em Python foi um exercício feito durante a aula de Introdução à Ciência de Dados para simular o monitoramento térmico de um satélite.



No script, apliquei lógica de programação para:



Validar as temperaturas e descartar "ruídos".



Gerar um gráfico da órbita usando a biblioteca matplotlib.



Classificar se o satélite está em estado crítico ou ideal.



Linguagem: Python.
